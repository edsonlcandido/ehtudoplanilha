Planilha Eh Tudo

Prints de Telas & Fluxos

Esta página apresenta capturas de tela (screenshots) e descrições dos principais fluxos exigidos para avaliação de uso de APIs Google (Sheets / Drive).

Observação: a cópia do template oficial e criação das abas necessárias ocorre automaticamente na página de configuração do app. Esta página serve apenas para demonstrar fluxos à avaliação das APIs Google.

0. Dashboard inicial

Link para a pagina de configuração.

1. Autorização OAuth

Fluxo de consentimento onde o usuário concede acesso apenas à planilha selecionada. Uso estrito dos escopos mínimos.

2. Configuração da Planilha

Usuário informa ou seleciona a planilha existente. Nenhuma alteração estrutural forçada.

3. Lançamento de Despesas

Interface enxuta para inserção rápida de linha (valor, categoria, descrição, data).

4. Resumo / Dashboard

Visualização agregada que consome apenas os dados já presentes na planilha do usuário.

5. Segurança e Revogação

O usuário pode efetuar logout e remover tokens; revogação adicional pode ser feita no painel Google.
